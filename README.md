🌦 Weather Dashboard
A simple and interactive weather dashboard that allows users to search for real-time weather information across different cities. Built with a focus on usability, clean design, and accurate data retrieval.

📌 Features
Search weather by city name
Display current temperature, humidity, and wind speed
Show weather conditions with icons (sunny, cloudy, rainy, etc.)
Responsive design for desktop and mobile
Easy to extend with additional APIs or features

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript 
Backend : Node.js / Express
API: Open-Meteo API
UI Design: Builder.io  integration for visual editing

🚀 Getting Started
Prerequisites
Node.js 
API key from Open-Meteo API
Installation
# Clone the repository
git clone https://github.com/vaishnavipurini-droid/weather.git

# Navigate into the project folder
cd weather

# Install dependencies
npm install

# Start development server
npm start
